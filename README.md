# M2 ó Image Scrambler

![version](https://img.shields.io/badge/version-0.1.0-blue.svg) ![python](https://img.shields.io/badge/python-3.7%2B-green.svg) ![license](https://img.shields.io/badge/license-MIT-lightgrey)

A small desktop application that demonstrates deterministic image scrambling and basic reversible transformations. The app exposes a three-step pipeline ó row/column permutation, pixel-level shuffle (FisherñYates or NumPy), and byte-level substitution ó and provides an interactive Tkinter GUI to scramble/unscramble images using an integer key.

---

## What this project does

`M2` provides a GUI utility (`M2.py`) that:

- Loads common image formats (PNG, JPG, BMP).
- Applies three reversible transformations to an image using a user-provided integer key:
  1. Row/column permutation (deterministic permutation from seed)
  2. Pixel-level shuffle (FisherñYates or NumPy permutation)
  3. Byte substitution (modular addition mod 256)
- Allows step-by-step scrambling and unscrambling with side-by-side previews of the original, scrambled, and unscrambled states.

See the main module: `M2.py`.

## Why this is useful

- **Educational**: Demonstrates deterministic permutations, shuffle algorithms, and reversible byte substitution.
- **Visual**: Immediate GUI feedback helps reason about how each transformation affects image data.
- **Reproducible**: Deterministic behavior using integer seeds makes it easy to reproduce and test.

This project is intended for learning and experimentation, not as production-grade cryptography.

## Key features

- Three selectable transformation steps with clear inverse operations
- Option to shuffle using manual FisherñYates or NumPy's built-in permutation
- GUI with algorithm log and three preview panes (Original, Last Scramble, Last Unscramble)
- Save the current working state to disk

## Quick start ó Get running

### Prerequisites

- Python 3.7 or newer
- Git (optional, for cloning)

### Clone the repository

git clone https://github.com/Znajka/M2.git
cd M2

### Install dependencies

pip install --upgrade pip
pip install opencv-python numpy pillow

### Run the app

python M2.py

The GUI will open. Use the controls to load an image, set the integer Key (k), choose a Step, then click `SCRAMBLE` or `UNSCRAMBLE`.

## Usage examples

- **Scramble rows with key `150`**:
  1. Load an image
  2. Select **Step 1: Row/Col Swap** and **Rows**
  3. Enter `150` in `Key (k)` and click `SCRAMBLE`

- **Shuffle pixels and undo**:
  1. Select **Step 2: Fisher-Yates Pixel** and choose algorithm
  2. Click `SCRAMBLE` to shuffle, then `UNSCRAMBLE` with the same key to restore

- **Apply substitution cipher**:
  1. Select **Step 3: Class A Substitution**
  2. Click `SCRAMBLE` (applies `(p + k) mod 256`) and `UNSCRAMBLE` to revert

## How it works (brief)

- Permutations are generated deterministically with `numpy.random.seed(key)` and either `numpy.random.permutation` or a manual FisherñYates implementation.
- The substitution step applies modular arithmetic to each byte value.
- All operations are reversible using inverse permutations or modular inverse (subtraction mod 256).

## Troubleshooting (short)

- If the app reports **Load an image first!**, click **Load Image** before other operations.
- If unscrambling does not restore the original, confirm you used the same key and the same step order when reversing.
